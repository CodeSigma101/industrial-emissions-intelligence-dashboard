# Industrial Emissions Intelligence Dashboard

An interactive data analytics platform exploring global industrial greenhouse gas emissions trends. Built with Python, Pandas, and Streamlit to demonstrate data cleaning, exploratory analysis, visualization, and corporate insight generation for sustainability-focused decision making.

## Core Features
* Executive Overview: Core metric performance indicators tracking total emission burdens.
* Emissions Trends: Interactive timeline trajectory analysis filtered by specific industrial activity.
* Country Comparison: Comparative historical groupings across tracked regional data boundaries.
* Sector Analysis: Burden analysis across steel, cement, and energy industries.
* Steel Industry Focus: Asset-level corporate tracker monitoring company-specific accountability and target alignments.
* Data Quality and Methodology: Complete transparency regarding data provenance, cleaning adjustments, and scoping boundaries.

## Technology Stack
* Application Framework: Streamlit
* Data Manipulation: Pandas
* Data Visualization: Plotly Express
* Development Environment: Python

## Installation and Local Deployment
1. Install dependencies:
   pip install -r requirements.txt
2. Execute application:
   streamlit run app.py
